## Diario de aprendizaje

### Día 1 - Fundamentos de entrada y salida

Hoy cerré las bases del curso con práctica directa en consola:

- Uso de `print()` para mostrar mensajes y resultados.
- Diferencia entre texto y operación real (por ejemplo, `"153 + 100"` se imprime literal si va entre comillas).
- Concatenación de cadenas con `+`.
- Secuencias de escape (`\n`, `\t`, `\"`, `\\`) para controlar formato del texto.
- Uso de `input()` para capturar datos del usuario y entender que devuelve texto.

Mini proyecto del día:

- Generador de nombres divertidos para cerveza combinando personaje + animal + color.
- Flujo completo: pedir datos, unirlos y mostrar un resultado final legible.

Sensación del cierre del día: base simple, pero clave para todo lo que viene.
