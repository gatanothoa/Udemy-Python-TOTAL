# Plantilla de actualizacion del README por dia

## Checklist rapido

- Actualiza avance consecutivo (X/16)
- Actualiza dias completados
- Actualiza dias restantes
- Actualiza barra de progreso
- Marca el proyecto del dia como completado en el mapa de niveles
- Agrega una seccion nueva en Diario de aprendizaje
- Actualiza Estructura actual
- Actualiza Proximo objetivo

## Formula de progreso

- Porcentaje = (dias_completados / 16) * 100
- Cada dia equivale a 6.25%

## Bloque para copiar en Diario de aprendizaje

### Dia X - Titulo del tema

Hoy avance en:

- Punto clave 1
- Punto clave 2
- Punto clave 3
- Punto clave 4
- Punto clave 5

Mini proyecto del dia:

- Que construiste
- Que practica reforzaste

Refuerzo aplicado:

- Ejercicios extra o examen del dia (si corresponde)

## Bloque para Estructura actual

- Dia X: tema 1, tema 2, tema 3, proyecto y recursos.

## Bloque para Proximo objetivo

- Completar Dia X+1 y pasar la barra a Y/16, manteniendo la racha.

## Tip de uso

Al terminar cada dia:

1. Abre este archivo.
2. Copia los bloques necesarios.
3. Pegalos en README.md.
4. Ajusta numeros y texto del dia.
5. Haz commit junto con los archivos del dia.
